Google Apps Script のためのユーティリティ関数とパターン.

Pattern #1

スプレッドシートの特定の範囲のデータを取得する 

function myFunction() {
  openSpreadsheetById("xxxxxxxx", (ss) => {
    openSheetByName(ss, "sheet 1", (sheet) => {
      openCsv(sheet, 1, 1, 3, 3, (csv) => {
        csv.forEach((row) => {
          console.log(row);
        });
      });
    });
  });
}

Pattern #2

スプレッドシートの特定の範囲のデータを書き換えるパターン 

function myFunction() {
  openSpreadsheetById("xxxxxxxx", (ss) => {
    openSheetByName(ss, "sheet 1", (sheet) => {
      changeCsv(sheet, 1, 1, 4, 4, () => {
        return [
          ["first name", "last name"],
          ["Maxine", "Ramsey"],
          ["Cecil", "Soto"],
          ["Tommie", "Bartlett"],
        ];
      });
    });
  });
}

Pattern #3

スプレッドシートの先頭から最終行までの範囲のデータを取得する 

function myFunction() {
  openSpreadsheetById("xxxxxxxx", (ss) => {
    openSheetByName(ss, "sheet 1", (sheet) => {
      openCsv(sheet, 1, 1, sheet.getLastRow(), sheet.getLastCol(), (csv) => {
        csv.forEach((row) => {
          console.log(row);
        });
      });
    });
  });
}

Pattern #5

スプレッドシートの最終行に新しいデータを追記する 

function myFunction() {
  openSpreadsheetById("xxxxxxxx", (ss) => {
    openSheetByName(ss, "sheet 1", (sheet) => {
      changeCsv(sheet, sheet.getLastRow(), 1, 1, 2, (csv) => {
        pushRow(csv, () => {
          return ["Harriett", "Brock"];
        });

        return csv;
      });
    });
  });
}
